Creación de virtual hosts informática y formación

Lo primero que debemos hacer es crear el primer fichero de nuestra web.

Creamos el directorio /var/www/informatica y alli el index.html.

Le damos contenido.

Como vamos a tener dos virtual host creamos también el directorio /var/www/formacion y su propio index.html.

Le damos contenido.

Ahora que ya tenemos algo que mostrar, en el directorio donde almacenamos los ficheros de configuración de Nginx, creamos el fichero de configuración tanto de informática como de formación.

Como vamos a cambiar la ruta de los logs de errores y acceso necesitamos crearlos.

Necesitamos que ambos ficheros tengan una configuración adecuada con las necesidades que tenemos para nuestros virtual hosts.

Es importante definir en que puerto escucha, el nombre que sirve, el fichero que va a mostrar y la localización de los ficheros de configuración.
